## Требования:
- Ruby: 3.1.3
- Node.js: v18.12.1
- Rails: 7.0.4
- Yarn: 1.22.19

## Запуск проекта

```bash
# Установка зависимостей
bundle install
yarn

# Прекомпиляция ассетов
bundle exec rails assets:precompile

# Создание и минимальное заполнение базы данных
bundle exec rails db:migrate
bundle exec rails db:seed

# Запуск сайта
bundle exec rails s
```
